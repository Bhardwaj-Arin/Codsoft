# CODSOFT Data Science Internship Portfolio

Welcome to my repository showcasing the projects completed during my Data Science internship at CodSoft. This repository contains end-to-end machine learning pipelines, data analysis, and predictive models implemented using Python and Jupyter Notebooks.

---

## 🚀 Projects Overview

### 1. Customer Churn Prediction
*   **Objective:** Predict whether a customer will leave a bank or subscription service based on their demographic and behavioral profile.
*   **Key Highlights:** Exploratory Data Analysis (EDA), handling class imbalance, feature scaling with `StandardScaler`, and training classification algorithms like `LogisticRegression` and `RandomForestClassifier`.
*   **Directory:** `Customer Churn/`

### 2. Movie Genre Classification
*   **Objective:** Predict the genre of a movie based on its textual description or plot summary.
*   **Key Highlights:** Natural Language Processing (NLP) techniques, text preprocessing, TF-IDF vectorization, and multi-class classification models.
*   **Directory:** `Movie Genre Classification/`

### 3. SMS Spam Detection
*   **Objective:** Build an AI model to classify SMS text messages as either "Ham" (legitimate) or "Spam".
*   **Key Highlights:** Text tokenization, data cleaning, processing imbalanced text data, and implementing classifiers like Naive Bayes or Decision Trees to maximize precision and recall metrics.
*   **Directory:** `Spam Sms Detection/`
